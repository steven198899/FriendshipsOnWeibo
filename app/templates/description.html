Instamap

Google Maps with Instagram


Instagram is an online photo-sharing, video-sharing and social networking service that enables its users to take pictures and videos, apply digital filters to them, and share them on a variety of social networking services.

Google Maps is a web mapping service application and technology provided by Google, that powers many map-based services, including the Google Maps website, Google Ride Finder, Google Transit, and maps embedded on third-party websites via the Google Maps API.

In our project, the following functions are contained.

We will show all the follows of a user first. Then, we can select one of them to find some of their pictures that posted with certain locations, and mark these locations on Google Maps. When we click the mark on the map, a info-window will pop up with detail messages of the picture.

In addition, we can query all the follows’ pictures in a certain city, which can be input by user. Besides, we can query the result in a certain distance around the city that the user input. For example, if the user input “Buffalo”, all the pictures posted by this user’s follows in Buffalo will be marked on the map. If the user input a concrete distance, 100 miles, then the results in this circle centered by Buffalo will be marked.
